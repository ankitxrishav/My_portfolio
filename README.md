## 🚀 Projects

### 📊 [Crimes Against Women – Analysis & Prediction](https://github.com/ankitxrishav/Crimes-Against-Women-Analysis-and-prediction)
Data-driven insights and predictive modeling to understand regional crime trends against women in India.
